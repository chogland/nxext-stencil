# Avatar: Overview

`Avatar` is a graphical representation of a user. It can contain the user's initials or an image of the user.

The avatar has been upgraded according to the Phase 2 experience.

Phase 2 Avatar does not support the _shadow_ property. All other functionality remains the same.
