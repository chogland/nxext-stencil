# Box: Overview

`Box` is a component used to add additional styles around and to slotted HTML elements and/or other RDS components.

The component is comprised of two nested inline DOM elements. The first inline element is there to add margin/padding to, handled by the _margin_ and _padding_ properties, and the second inline element is there for the addition of any custom CSS passed in through the _styles_ property.

You can add as many RDS components to the slot as you would like. Below is an unstyled box component that houses a button component. You may find yourself adding box components around any sub components that you would like to add padding/margin to, for example.

<Canvas withSource="none">
  <rds-box>
    <rds-button>I'm a button inside of a box.</rds-button>
  </rds-box>
</Canvas>

```
<RdsBox>
  <RdsButton>I'm a button inside of a box.</RdsButton>
</RdsBox>
```

Box has properties added to manipulate its margin, padding, text color, background color, and a catch-all _styles_ property that can add custom CSS directly to it. See each property explained in detail below.

## Box Options

### Display

The `display` property adjusts the CSS display styles applied to the box. There are only three options to choose from: _block_, _inline-block_ and _inline_. It is set to **block** by default.

The display property does not have flex support built in because the `RdsFlex` component is available to handle all flex options. Alternatively, raw CSS can be passed into box's _styles_ property.

Below is an example of two box components, each with their default display value of _block_.

### Color

The `color` property applies a text color to the box component and its slotted components. If the slotted components inherit its parent's text color, the text color will change when this property is updated.

You can find the list of colors supported with the box component on our [Brand Identity/Colors page](/story/brand-identity-colors--page).

Below is an example of the box component manipulating a slotted text component's color.

<Canvas withSource="none">
  <rds-box color="purple-600">
    <rds-text>I'm a text component inside of a box.</rds-text>
  </rds-box>
</Canvas>

```

</RdsBox>
```

## Usage Guidelines

> **Use custom styles sparingly**

This component was created to allow you to make occasional sizing/alignment exceptions to your components. The point is not to re-style any RDS components. Please work with your team's designer on any custom styling needs.

> **Ensure color choices are accessible**

If you are manipulating background or text colors, please reference the [Accessibility/Color Contrast page](/story/accessibility-color-contrast--page) to ensure your color choices are ADA compliant.
