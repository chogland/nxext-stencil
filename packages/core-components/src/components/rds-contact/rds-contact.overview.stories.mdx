# Contact: Overview

`Contact` is a simple pattern that includes avatar and text components to offer up a contact component that accepts Name, Role and another line of details.

## Usage Guidelines

> Informational

Use `Contact` to display one or more persons details like name and position.

> Interactive

Optionally add buttons to interact with `Contact` to assist a user in e-mailing, phoning or other contact methods.
