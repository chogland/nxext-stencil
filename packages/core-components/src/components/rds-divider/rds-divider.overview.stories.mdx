# Divider: Overview

`Divider` is a horizontal or vertical rule used to group content in lists and layouts.

## Variations

### Default

By default, the divider spans horizontally and has a size _medium (24px)_ margin applied to its top and bottom.

You can change the direction of the divider from horizontal to vertical by changing the `type` property to _true_.

## Usage Guidelines

> **Use secondary to white space**

Use a divider if elements cannot be separated appropriately with white space alone.

> **Use infrequently**

Use dividers sparingly, to create groupings rather than separate items. Frequent use of dividers is a jarring experience for the end-user.
