# Filter: Overview

## Filter

`Filter` is a container component that provides wrapper support for grouped form elements i.e. radio and checkbox input components needed for filtering page content.

## Usage Guidelines

The `Filter` component is to be used with `RdsButton` and can be applied by adding the `isFilter` attribute. By adding the `isFilter` attribute to the `RdsButton` it enables a toggle to communicate with the `Filter` component.
