# Checkbox Group: Overview

`CheckboxGroup` is a wrapper component that is used for grouping checkbox components. It provides a group label and a slot for adding as many checkboxes as needed. The checkboxes receive additional styling when they are part of the checkbox group component. The component allows you to easily pass the same `name` and `type` properties to all slotted checkboxes as well.

## Usage Guidelines

> **Be Concise**

The grouping component labels should be kept to a single line of text.

> **Use sentence case**

Labels should always be in sentence case. Capitalization should never be used to give more prominence to a specific element.

> **Don't override color**

Do not use custom colors for the grouping label or any sub-elements of checkbox group. The colors have been designed to be consistent and accessible.
