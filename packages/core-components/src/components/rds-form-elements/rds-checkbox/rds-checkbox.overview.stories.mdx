# Checkbox: Overview

`Checkboxes` are used for multiple choices, not for mutually exclusive choices. Each checkbox works independently from other checkboxes in the list, therefore checking an additional box does not affect any other selections.

## Usage Guidelines

> **Multiple Selections**

Use a checkbox when a user can choose more than one option in a form.

> **Be Concise**

The form element labels should be kept to a single line of text.

> **Use sentence case**

Labels should always be in sentence case. Capitalization should never be used to give more prominence to a specific element.

> **Don't override color**

Do not use custom colors for checkboxes. The colors have been designed to be consistent and accessible.
