# Date Picker: Overview

The `RdsInputDatePicker` is a component that combines an `RdsInput` with a `RdsDatePicker` to create an experience where a user can select a date by picking a date in a picker instance where it is displayed in an input.

The current supported date format for this component is DD/MM/YYYY: 20/12/2022.
