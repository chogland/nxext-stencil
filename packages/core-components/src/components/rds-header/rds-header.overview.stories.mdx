## NEW: Global Header

The new global header is intended for our upcoming Phase 2 experience. Please only use the _base_ `RdsHeader` with the _base_ `RdsAppLayout`.

You can see this in action in the _base_ [App Layout](?path=/story/patterns-app-layout-stories--base) story.

Slots have been provided for: `logo`, `search`, and `profile`

The `Logo` slot only displays on mobile screens. On all other screens, the logo should appear inside of the sidebar.
