# Icon: Overview

**Please see the [`RdsHeroIcon` component](?path=/story/components-hero-icon-overview--page).**

`Icons` are visual symbols used to represent ideas, objects or actions. They communicate messages at a glance, afford interactivity and draw attention to important information.

## Sizes

`Icon` sizes range from small to large.

## Colors

You can provide any color in the RDS color palette to the icon to change its color. _If a color isn't provided it will default to #000 or inherit from it's parent_
