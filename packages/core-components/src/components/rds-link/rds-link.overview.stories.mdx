# Link: Overview

`Link` is a traditional `<a>` tag, or a `<button>`, which points a user to another location. `RdsLink` is a component that utilizes both types of hyperlinks and is bootstrapped with a simple text-based style.

## Usage Guidelines

> **Be Concise**

Link text should be concise: 1 or 2 words, no longer than 4 words, with fewer than 20 characters including spaces. Don't use punctuation marks such as periods or exclamation points within the link.

> **Use sentence case**

Link text should always be in sentence case. Capitalization should never be used to give more prominence to a specific link.

> **Use icons for clarity**

Icons can be used in links when additional clarity is required and the icon is highly relevant to the action. Icons should not be used for decoration.
