## Usage Guidelines

> **Use simple/semantic lists properly**

Use ordered lists for sequences and unordered lists in other contexts.

> **Use a list to show multiple list items**

Lists aren't intended to show only one item, as a single item isn't really a list. If you want to set a single item off from surrounding text, then use other formatting with a HTML div, span, or RdsBox element.

> **Should I use a description/stacked list or a table?**

Tables and lists are both ways to present a set of similarly structured items. As a general rule, when each item has three or more pieces of related data, its recommended to use a table.
