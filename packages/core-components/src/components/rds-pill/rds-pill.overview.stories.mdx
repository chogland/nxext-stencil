# Pill: Overview

The `Pill` component is used to indicate the status of an active process.

## Pill

The `Pill` component is comprised of a status `Indicator` and a label.

## Usage Guidelines

> Indicator

The status indicator should accurately reflect the current status of the process and automatically update to reflect status changes.

> Pill

The Pill is not a button and should not be used as an interactive element.

> Color

Do not change the background color of the pill.
