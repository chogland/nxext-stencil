# Progress Circle: Overview

## Progress Circle

A `Progress Circle` includes a circular track that fills to show the progression of a task with a known duration. Progress circles are not interactive. The value is reflected as a label in the middle of the component.

## Usage Guidelines

> Always report progress accurately.

> Only use progress bars for tasks that are quantifiable.

> Use progress bars for tasks with a well-defined duration.
