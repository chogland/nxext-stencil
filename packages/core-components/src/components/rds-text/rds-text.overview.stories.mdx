# Text: Overview

`Text` creates a pre-styled `<p>` tag to use for text blocks.

Setting the _type_ to "inline" will render a `<span>` instead. This is useful in wrapping smaller pieces of text.

### Font Weight

To change the font-weight CSS property, update the `weight` prop to `normal` (default & 400), `medium` (500), `semibold` (600), `bold` (700), or `extrabold` (800).

### Font Size

To change the font-size CSS property, update the `size` prop to `base` (default & 1rem), `xxs` (.5rem), `xs` (.75rem), `sm` (.875rem), `lg` (1.25rem), `xl` (1.5rem), `2xl` (1.875rem), `3xl` (2.25rem), `4xl` (3rem), `5xl` (3.75rem), `6xl` (4.5rem), or `7xl` (6rem).

To change the top and bottom margin CSS properties, update the `spacing` prop to `none` (0px), `sm` (8px), `md` (16px), or `lg` (24px).

## Usage Guidelines

> Custom styles

The new RDS font, Inter, is automatically applied to the `RdsText` component, as well as pre-defined rules for spacing, font-weight, and font-size. You should not add any custom style within the component, **unless**, you want to add a color to the text. In that case, refer to the 'With Custom Color' story above to see how to add a span inside of the component for the acheived look and feel.
